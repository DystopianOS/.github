# Welcome to DystopianOS

In a world of decaying legacy systems and corporate surveillance, we build tools for resistance, privacy, autonomy, and controlled chaos.

We create open-source software that empowers individuals against centralized control — secure communication, offline-first apps, adversarial resilience, and systems designed to outlast collapse.

## Core Principles
- **Minimalist & hardened** — less surface area for exploits.
- **Audit-friendly** — readable code > clever code.
